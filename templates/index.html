Public Domain Reading and Annotation

As a demo for Books in Browsers 2013, FuturePress has created an open annotation space for public domain books from Project Gutenberg. Upload a book from Project Gutenberg or simply paste the link to the book. You'll be able to read the book using Epub.js and annotate with Hypothe.is.

We hope you'll share our excitement for making books first class web resources.
